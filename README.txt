# social
social network on the Yii2 framework

Использовались модули OpenSerwer:
HTTP: Apache-PHP-7;
PHP: PHP-7.1;
MySQL/MariaDB: MariaDB-10.1;

На данный момент работают страницы:
-регистрация,
-авторизация,
-восстановление пароля по email,
-профиль,
-друзея,
-новости,
-галерея.

Можно:
-загружать и удалять изображения,
-создавать и удалять новости и комментарии,
-оценивать записи,
-редактировать профиль,
-добавлять и удалять подписки,
-следить за новостями подписок.
